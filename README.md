Dynamic Interactive Resume Builder
A powerful, real-time resume builder that lets you create professional resumes with live preview, multi-step form inputs, custom styling, and print-ready output.

✨ Features
Live Preview – See changes instantly as you type
Multi-step Form – Organize personal, experience, education, and projects sections
Project Management – Add projects with name, description, tech stack, and duration
Custom Styling – Choose fonts and accent colors
Print-Ready – Clean @media print styles (no timestamp, app name, or borders)
Responsive – Works on all devices

No UI elements (forms, buttons, etc.)
No app name or timestamp
No decorative borders
